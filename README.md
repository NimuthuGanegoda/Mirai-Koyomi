# Sri Lanka Holidays Master (2021-2028)

![Sri Lanka Holidays](https://img.shields.io/badge/Sri_Lanka-Holidays-blue?style=for-the-badge)
![Apple Calendar Compatible](https://img.shields.io/badge/Apple_Calendar-Compatible-success?style=for-the-badge)

A comprehensive, production-ready collection of Sri Lankan public, bank, and mercantile holidays. This repository integrates data from multiple sources to provide the most feature-rich holiday dataset available, complete with visual markers and smart alarms.

## Features

- **Extended Coverage**: Full support for years **2021 through 2028**.
- **Official Markers**: Includes traditional markers in summaries:
  - `*` Public Holidays
  - `†` Bank Holidays
  - `‡` Mercantile Holidays
- **Smart Alarms**: Integrated `VALARM` set for 9:00 AM the day before each holiday.
- **Multiple Formats**: Available in **JSON, ICS, CSV, and XML**.
- **Master Subscription**: One-click subscription to `ics/srilanka-holidays.ics` for all years.

## Quick Start (Apple/Google Calendar)

To subscribe to the master calendar:
1. Copy the URL to the raw `ics/srilanka-holidays.ics` file.
2. In your calendar app, select "New Calendar Subscription".
3. Paste the URL and enjoy automatic updates.
